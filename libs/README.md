# USING WITH DARKET
Using python with the darknet shared library results much easier than trying to use opencv functions that
use Yolov4 tiny, it also gives better results as we are using the same libary used to train the model

Please [Download and build darknet](https://github.com/pjreddie/darknet) without opencv support and, preferably with CUDA
support and paste libdarknet.so in this directory before running anything
