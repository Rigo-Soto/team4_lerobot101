# USING WITH DARKET
Using python with the darknet library results much easier than trying to use opencv functions that
use Yolov4 tiny, it also gives better results as we are using the same libary used to train the model

Please [Download and build darknet](https://github.com/pjreddie/darknet) with opencv support and, preferably CUDA support
before running anything and paste libdarknet.so in this directory before running anything
